Práctica 1: Imágenes optimizadas para la web

Introducción

En esta práctica, trabajarás con imágenes optimizadas y un logotipo SVG. Completa la investigación de licencias y prepara tus recursos.

Banco de recursos

Banco de recursos Tipo de licencia
Pexels Pexels permite la descarga, utilización y modificación de sus imágenes de forma gratuita sin la necesidad de dar créditos bajo ciertas condiciones:
  • Las personas identificables en la foto no pueden salir desfavorecidas ni de manera ofensiva
  • No se puede vender las fotos sin su previa modificación
  • No se pueden usar las imágenes de forma que parezca que las personas o marcas de las imágenes estén promocionando tu producto
  • No se puede redistribuir las imágenes en otros bancos de recursos
  • No se puede usar las fotos como parte de tu marca comercial
Pixabay Pixabay permite utilizar su contenido de forma gratuita, sin atribuir créditos y la modificación de los mismos con algunas restricciones:
  • No se puede vender ni distribuir el contenido sin haber agregado un esfuerzo creativo, es decir, tienes que utilizar el contendio para algo, no se redistribuir sin acompañar
  • Si el contendio tiene alguna marca o logo reconocible, no se podrá utilizar con propósitos comerciales
  • No está permitido el uso ilegal o inmoral del contenido
  • No se permite el uso del contenido de manera engañosa
  • No se puede utilizar el contendio como parte de tu marca personal
Freepik Freepik ofrece diferentes tipos de licencia según si eres usuario gratuito o tienes suscripción, con sus propias condiciones:
  • Licencia gratuita (Free, con atribución):
    • Permite usar el contenido para fines personales o comerciales, siempre que des crédito (“designed by Freepik”).
    • Se puede modificar el contenido (obras derivadas).
    • No se puede sublicenciar, vender o alquilar el recurso original ni modificado.
    • No se puede distribuir como recurso para otros bancos sin autorización.
  • Licencia Premium (sin atribución):
    • No es necesario dar crédito si tienes suscripción Premium, Premium+ o Pro.
    • Derecho de uso no exclusivo, no transferible y no sublicenciable.
    • Puedes usar el contenido de forma ilimitada (según los usos permitidos) y crear obras derivadas.
    • No puedes revender el contenido en su forma original ni permitir que terceros lo descarguen como recurso para redistribución.
  • Licencia de Merchandising (solo para suscriptores Pro):
    • Permite usar contenido de Freepik para productos físicos con fin comercial (ropa, tazas, pósteres, etc.).
    • Hay un límite: máximo **100.000 unidades** por recurso individual.
  • Licencia para fuentes (“Licensed Fonts”):
    • La licencia es no exclusiva, revocable y no es transferible.
    • Puedes usar las fuentes en tus documentos o sitios web, pero si incrustas la fuente, debe ser un subconjunto y protegerse para que no se extraiga fácilmente.
    • No puedes distribuir, sublicenciar ni modificar las fuentes por sí solas (solo en un documento).
  • Licencia específica (“Specific License”):
    • Algunos recursos pueden tener una licencia distinta a las anteriores (“Licencia Específica”).
    • Cuando exista, prevalecen las condiciones de esa licencia específica sobre las generales.

Comparación de imágenes

Formato Peso medio Soporte de transparencia Escalabilidad Casos de uso recomendados
JPG 5MB No No Fotografías, contenido para web, redes sociales, almacenamiento de grandes colecciones de fotos
PNG Más pesados que JPG Sí No Imágenes con transparencia, Imágenes de alta calidad, edición y trabajo profesional
SVG 10Kb - 50Kb, Ligero en general Sí Sí Logotipos, iconos, marcas...
WebP En general más ligeros que JPG y PNG Sí No Web y apps, animaciones, reducción de tamaño sin sacrificar calidad
